# Rebel Ants Playground — Complete v2

Includes:
- Ant Tunnel Forage (provably fair)
- Queen's Egg Hatch (jackpot odds + crate art)
- Tailwind styling

## Quickstart
```bash
cp .env.example .env.local
# edit .env.local: set SERVER_SEED to a long random string
npm i
npm run dev
# open http://localhost:3000
```

Drop your crate PNGs into `/public/crates/` as `common.png`, `rare.png`, `ultra.png`.
